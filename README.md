# Weather-Forecasting
Streamlit App : https://weather-forecasting-002586.streamlit.app/
Live Data Project

🌦️ **Weather Forecasting using RNN & LSTM (Live Data Based)**

I built a Weather Forecasting System that not only predicts the next 10 days but is also **based on real-time (live) weather data**.

🔹 **What this project does:**

* Predicts Temperature, Humidity, Pressure, Wind Speed & Rainfall
* Uses **live weather data from API** for current conditions
* Generates **10-day future forecasts** using deep learning

🔹 **Tech Stack:**

* Python, Pandas, NumPy
* TensorFlow / Keras (RNN + LSTM)
* Streamlit (interactive UI)
* Open-Meteo Historical Data (live data)

🔹 **Key Highlights:**

* Combination of **real-time data + time series forecasting**
* Multi-feature prediction (not just temperature)
* End-to-end ML project (training → prediction → deployment)

🚀 This project helped me understand how to integrate **live APIs with deep learning models** for real-world applications.
